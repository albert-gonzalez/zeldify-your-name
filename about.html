About Zeldify Your Name

Zeldify Your Name is a simple application that convert the input string in a name of 'The Legend of Zelda' games. The really interesting is the used technologies.

The main technologies are Requirejs, AngularJs, Bootstrap and JQuery. KarmaJS and Jasmine are used for Unit Testing. Protractor,Selenium and Jasmine are used por e2e Testing. Bower and npm are used for package management and Grunt for building the css and js distribution.
